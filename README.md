# iCPU

iCPU is an educational CPU project built from the ground up using **iHDL**, a hardware description language and simulator written in Go.

The project follows the philosophy of **Nand2Tetris**, but instead of starting from a NAND gate, the hardware is constructed from the primitive logic gates:

- AND
- OR
- NOT

The goal is to understand how a complete computer is built from simple logic gates, one abstraction layer at a time.
